Click button set by data-sc-click

Code

Html

<input type="button" data-sc-id="Button" data-sc-click="this.set('text',"etet")" data-bind="value:text, click: click" class="sc-button"/>

Javascript

_sc.Factories.createBaseComponent({
  base: "ButtonBase",
  name: "button",
  selector: ".sc-button",
  attributes: [ 
    { name: "text", value: "Click me" }
  ]
 });

  //Exposing to global window
  window.app = _sc.Factories.createApp();

Result

What you need to know.

  • Base

    The base property is used to defined which base class you want to extend. This will use the extend method from backbone.

  • data-sc-click

    In order to be able to write the behavior you want in you item, we have provided you a click property. This will generate the data-sc-click

    If the control is a type button, you will have the click automatically register for you.

  • The syntax

    The click will be set by the Item and will execute the Invocation pipeline. Here the this is the model.
