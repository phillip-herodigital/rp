Demo: The Component in sitecore.

Component has nothing related with Backbone or KO. It is a custom function made by Sitecore.
The idea is to stack Compoments in the SPEAK JS Framework. The compoment will have the information necessary to create a Control (see later). For now just keep in mind that a Component does not add behavior, it is just a definition. Please, the core could have componet ready to use for you. This means, in this example, you could have more than 2 components at the end.

1) Creating a components

Open your console


  alert("Core has "+ _sc.Components.length + " already in the system");

  //we define a model
  var Button = _sc.Definitions.Models.Model.extend({
    initialize: function() {
      this.set("FromInput", "");
    }
  });

  //we define the View
  var ButtonView = _sc.Definitions.Views.View.extend({
    initialize: function() {
      //after 2 seconds, change the value
      var view = this;
    }
  });

  //create the component
  _sc.Factories.createComponent(/*name*/"button", Button, ButtonView, /*class*/ ".btn");

  //and now we see if the component is stacked in Sitecore SPEAK JS.
  console.log("This component has been added");
  console.log(_sc.Components)

  alert("You have now "+ _sc.Components.length + " one component ");
