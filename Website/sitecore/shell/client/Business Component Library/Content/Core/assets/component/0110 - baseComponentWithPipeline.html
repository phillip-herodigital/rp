Simple text control and run method

Code

Html

<span data-sc-id="SimpleTextControl" data-bind="text:text" class="sc-text"></span>

Javascript

_sc.Factories.createBaseComponent({
  name: "TextControl",
  selector: ".sc-text",
  attributes: [ 
    { name: "text", value: "$el.text" }
  ],
  beforeRender: function() {
    alert("Just before rendering, the model and the view have been instantiated and the bindings have been applied");
  },
  render: function() {
    alert("Generally you use render when you use Javascript Templates");
  },
  afterRender: function() {
    alert("Now you are sure that the HTML is present for the Component");
  }
 });

$("#run").click(function(){
    //exposing to global for documentation purpose
    window.app = _sc.Factories.createApp();
});

Result

Run the app manually to see the Pipeline in action

What you need to know.

  • _sc.Factories.createApp()

    When you execute this method, it will initialize all the Controls, after executes all beforeRender methods, all Render methods and all the afterRender methods.
