More complex component

Code

Html

<input type="button" data-sc-id="complex" data-bind="value:text" class="sc-complex"/>

Javascript

  var model = Sitecore.Definitions.Models.ButtonBaseModel.extend(
  {
    initialize: function (options)
    {
      this._super();
      this.set("text", "Set from the model");
    }
  });

  var view = Sitecore.Definitions.Views.ButtonBaseView.extend(
    {
      initialize: function(options)
      {
        this._super();
      },
      events: {
        "click": "doSomething"
      },
      doSomething: function() {
        alert("he ho");
      }
  });

  _sc.Factories.createComponent("complex", model, view, ".sc-complex");

  //Exposing to global window
  window.app = _sc.Factories.createApp();

Result

What you need to know.

  • createComponent

    When you want to create more complex controls, you can do it by calling the createComponent. There you will define the Model, the View by your self. Please note, only the model will be exposed at the end.

  • this._super

    Each object (models and views) has a _super funtion in order to call their base control.
