Another kind of initialization

Code

Html

<input type="checkbox" data-sc-id="Checkbox" checked="checked"  data-bind="checked:isChecked" class="sc-checkbox"/>

Javascript

_sc.Factories.createBaseComponent({
  name: "Checkbox",
  selector: ".sc-checkbox",
  attributes: [ 
    { name: "isChecked", value: "$el.prop:checked" }
  ]
 });

  //Exposing to global window
  window.app = _sc.Factories.createApp();

Result

What you need to know.

  • Another kind of initialization

    This example will show you how to initialize your model based on more complex case. Here this.$el.prop("checked") will be called.

    You could also do value:"$el.attr:class" or "$el.data:sc-some-special-stuff"
