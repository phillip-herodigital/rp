Create a simple text control

Code

Html

<span data-sc-id="SimpleTextControl" data-bind="text:text" class="sc-text"></span>

Javascript

_sc.Factories.createBaseComponent({
  name: "TextControl",
  selector: ".sc-text",
  attributes: [ 
    { name: "text", value: "Hello, world!" }
  ]
 });

window.app = _sc.Factories.createApp();

Result

What you need to know.

  • data-sc-id attribute in the html

    this is the name of you control.

  • name

    this is the name of your Component (a control is built based on the component). Component can be seen as a Class, control can be seen as an object.

  • css selector

    Will indicate which html element correspond to which Component.

  • attributes

    An array of object to set the proper value to your model. Currently you have name (name of the attribute) and value

  • What a control looks like
    • name: comes from data-sc-id
    • model: this is your model (a backbone model)
    • view: this is your view (a backbone view)
    • collection: optionnal, a list of model
  • What is exposed

    The app is not exposed to the global Object. It is only available on your PageCode (you will see that later). This is why I am assigning the app to window.

    Open your console type: app.SimpleTextControl and press return

    In order to levarge binding, cross-binding and events deletagtion, we decided to only exposed the model on the app object.

  • Try to type this in your console

    app.SimpleTextControl.get("text")
